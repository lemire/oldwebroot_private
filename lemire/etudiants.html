Financement

Si vous avez besoin d'appui financier en tant qu'un de mes étudiants, j'exige que vous fassiez un effort soutenu pour obtenir du financement d'organismes externes.

Si vous n'êtes pas canadien, vous devriez consulter le programme québécois de bourses d'excellence pour les étudiants étrangers. Ce concours est ouvert à tous les ans. Il inclut des bourses pour les séjours de courte durée (4 mois), les stages post-doctoraux, etc. Sachez ensuite qu'il y a un programme d'exemption des droits de scolarité pour les étudiants étrangers. La cotutelle est une option que vous devez aussi considérer. Dans ce modèle vous pouvez être étudiant dans deux universités.

Si vous êtes canadien, il y a plusieurs programmes de bourses :

  • programmes d'aide financières du gouvernement du Québec;
  • programmes de bourses du FQRNT (incluant bourses pour étudiants étrangers);
  • programmes de bourses du CRSNG;
  • Agence Universitaire de la francophonie (programmes de bourses).
