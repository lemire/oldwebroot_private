Sokoban/Sobokan for PalmOS in Java

by Daniel Lemire

Sokoban/Sobokan pour PalmOS en Java

par Daniel Lemire

It is a very simple game. You have to move the three keys on the three keylocks by moving the little fellow. You move him by clicking on the screen in an adjacent tile. For example, if you click in the spot to left of the little guy, he will move left. You push keys by clicking on them when the little man is right by the key, assuming there is free room behind the key.

Il s'agit d'un jeu en Java dont les règles sont simples. Il faut placer les trois clefs sur les trois serrures en déplaçant le petit personnage. On le déplace en cliquant sur l'écran avec la souris sur une case adjacente. Par exemple, si vous cliquez sur un espace libre juste à gauche du bonhomme, il se déplacera vers la gauche. On déplace les clefs en cliquant sur elles, en supposant que le petit homme est à côté de la clef et qu'il y a de l'espace derrière la clef.

Pour le programmeur! Sources de la version 1.0 (10Ko) (mise à jour le 20 février 2002) : exige Superwaba 1.x.

To play on your Palm or Visor now! Download the compiled bytecode version for PalmOS, version 1.02 (63 KB). It includes a version of Superwaba.

If you have another type of device such as WinCE, you should be able to play as well, but you'll need to get the corresponding Superwaba JVM (see http://www.superwaba.org) instead of the provided PalmOS version.

Pour jouer sur votre Palm ou Visor! Téléchargez la version compilée en bytecode pour PalmOS, version 1.0 (63 Ko). J'inclus une version de Superwaba.

Si vous avez un autre type d'appareil tel que WinCE, vous devriez pouvoir jouer tout de même, mais vous allez devoir remplacer la machine virtuelle (JVM) PalmOS par une autre correspondant à votre appareil (voir http://www.superwaba.org).

To play on your Palm or Visor now! Download the compiled native version for PalmOS, version 1.0 (faster and only 24KB) : this faster version was done by George Ruban, see his web page where he described how he did this using Jump.

Pour jouer sur votre Palm ou Visor! Téléchargez la version compilée native pour PalmOS, version 1.0 (plus rapide que le bytecode et seulement 24Ko) : cette version plus rapide fut réalisée par George Ruban, voir sa page web où il décrit comme il s'y est pris en utilisant Jump.
