Projets de premier cycle en informatique

  • En Java, la classe String est munie d'une fonction de hachage basée sur le nombre 31. Est-ce que la valeur exacte de ce nombre (31) est important?
  • Les sites web de commerce électronique sont constitués de plusieurs pages (parfois des milliers ou des millions). En considérant un tel site web comme un graphe (en utilisant les liens entre les pages comme des liens entre les noeuds), que peut-on dire sur la «topologie» de ces sites web? Est-ce qu'on peut, par exemple, en fait une représentation graphique utile?
  • Le site web DBLP fourni une base de données permettant de consulter les publications de milliers de chercheurs. On peut considérer que deux chercheurs sont des voisins s'ils ont une publication commune. De voisin en voisin, on peut passer d'un chercheur à un autre. Est-ce qu'on peut créer un outil logiciel qui calcule rapidement la distance moyenne entre deux chercheurs? Quelle est la distance maximale entre deux chercheurs?
  • Les courriels comprennent souvent un «épilogue» ou une signature qui revient constamment d'un courriel à un autre. Est-ce qu'on peut développer un outil qui reconnaît de manière efficace les épilogues, pour éventuellement les faire disparaître? Données: Enron Data Set.
  • Il existe plusieurs algorithmes pour permuter aléatoirement un ensemble («brasser les cartes»). On dit qu'un tel algorithme n'a pas de biais s'il peut générer n'importe quelle permutation possible avec la même probabilité. Est-ce qu'on peut développer un outil qui vérifie si l'algorithme de permutation est sans biais?
