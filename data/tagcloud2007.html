Tag-cloud data

We constructed two data sets to experiment tag-cloud drawing. They should be suitable for any basic tag-cloud drawing optimization work. The two data sets were generated in January 2007. The pixel width and height of tags were computed using Mozilla firefox and a certain CSS file (not available here). The data sets are available for download as CSV files using the format "name of tag", "tag weight", "width (pixels)", "height (pixels)".

  • GutenbergTagClouds.zip: this data set is made of 80 CSV files. It was generated from Project Gutenberg ebooks.
  • ZoomClouds_29jan2007.zip: this time, we have 65 CSV files generated from data made available at the ZoomClouds Web site.

Related paper:

  • Owen Kaser and Daniel Lemire, Tag-Cloud Drawing: Algorithms for Cloud Visualization. In Tagging and Metadata for Social Information Organization (WWW 2007), 2007.
